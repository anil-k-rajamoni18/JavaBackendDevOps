Centralized logging stacks capture, parse, and store logs from all Kubernetes pods.

### ğŸ”§ Components:
- **Fluentd**: Log aggregator
- **Elasticsearch**: Search and storage engine
- **Kibana**: Visualization UI

### ğŸ“¸ Fluentd DaemonSet Example:
```yaml

kind: DaemonSet
metadata:
  name: fluentd
spec:
  selector:
    matchLabels:
      name: fluentd
  template:
    metadata:
      labels:
        name: fluentd
    spec:
      containers:
      - name: fluentd
        image: fluent/fluentd:v1.14-1
        volumeMounts:
        - name: varlog
          mountPath: /var/log
      volumes:
      - name: varlog
        hostPath:
          path: /var/log
```
